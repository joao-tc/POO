Trabalho Final - Programação Orientada a Objetos - Turma 12

Como utilizar o programa:

1) Arquivos de texto:
    Os objetos que serão utilizados durante a operação do programa são criados com base em arquivos CSV que podem ser achados na pasta do código fonte.
    A explicação mais detalhada de como modificar os arquivos é:

    1.0) Para o arquivo "Garagem.txt", o padrão deve ser o seguinte:
        tipo(1 para locomotiva e 0 para vagão);ID;Carga máxima(em toneladas)
        Exemplo: 1;2;20 é uma locomotiva com ID = 2, que carrega até 20 toneladas

    1.1) Para o arquivo "Composicoes.txt", o padrão deve ser o seguinte:
        ID da composição;tipo(1 para locomotiva e 0 para vagão);ID do carro
        Exemplo: 0;1;0
                 0;0;0
                 0;0;1 é uma composição com 1 locomotiva (id = 0) e 2 vagões (id = 0 e id = 1)

2) Não deixe linhas em branco;

3) As demais interações com o programa devem ser feitos pelo console.
